About

We bought a house so that we would have more space to live, and to do projects, have pets, make changes, and generally have more control over our living situation.

We decided to buy a house which needed a lot of cosmetic work so that we could try new things and learn how to do home improvement. This blog shows the projects we've been working on, and maybe some tips and tricks if you want to try this stuff yourself.

Technical Details

The site was designed and written in AngularJS, a nifty framework which binds javascript and html variables together. The code, if you want to use it, is available on github. Please give us credit if you use it!

This blog is currently running off of a raspberry pi sitting on the granite floor of our living room.
